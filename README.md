# Verdure — Ethical Fashion AI Assistant 🌱

**Verdure** is an AI-powered, ethical fashion advisory web application built for the modern, eco-conscious consumer. Built entirely as an elegant single-page application (SPA), it provides real-time, conversational guidance on sustainable fabrics, ethical brands, greenwashing, and mindful shopping.

## 🚀 Features
- **Real-Time Fashion AI:** Powered by the Google Gemini Flash API to answer domain-specific questions about sustainability.
- **Domain-Locked:** The AI is carefully prompted and restricted to *only* answer fashion, sustainability, and garment care-related questions.
- **Quick Topic Cards:** Interactive quick-start chips for common topics like "Capsule Wardrobes", "Certifications", and "Greenwashing".
- **Premium UI/UX:** A stunning dark-mode interface featuring glassmorphism, pulse animations, staggered typing indicators, and hover-triggered dynamic social tooltips.
- **Zero Dependencies:** Pure HTML, CSS, and Vanilla JavaScript. No bundlers, npm packages, or framework setup required.

## 🛠️ Technology Stack
- **Structure:** HTML5
- **Styling:** CSS3 (Custom Variables, Keyframe Animations, Flexbox, UI Effects)
- **Logic & API:** Vanilla JavaScript (ES6+)
- **LLM Integration:** Google Gemini REST API (`gemini-flash-lite-latest`)
- **Typography:** Google Fonts (Dosis)

## 📥 How to Run Locally
Since the project uses zero external dependencies, it will run instantly in any modern web browser. 

1. Clone this repository:
   ```bash
   git clone https://github.com/Devanshu-Dhyanu/Ethical-Fashion-Assistent.git
   ```
2. Get your free Gemini API key from [Google AI Studio](https://aistudio.google.com/).
3. Open `ethical-fashion-assistant.html` in a code editor, locate the JavaScript section at the bottom, and insert your API key:
   ```javascript
   const API_KEY = 'YOUR_FREE_GEMINI_API_KEY_HERE';
   ```
4. Double click `ethical-fashion-assistant.html` to open it in your standard web browser.
